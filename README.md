# 幸福路的火锅店

火锅店模拟经营游戏 - HTML5 Canvas 版

## 运行方式

直接在浏览器中打开 `index.html` 即可开始游戏。

## 游戏玩法

### 核心循环
1. **顾客进店** - 顾客会自动来到火锅店，选择空桌坐下
2. **点单** - 顾客坐下后会自动点餐（锅底+配菜）
3. **烹饪** - 厨房自动开始烹饪
4. **上菜** - 烹饪完成后自动上菜
5. **用餐** - 顾客开始用餐
6. **结账** - 用餐完成后点击桌子进行结账
7. **打扫** - 结账后桌子变脏，点击桌子打扫

### 界面说明
- **顶部栏**：显示金币💰、钻石💎、天数
- **底部按钮**：
  - 菜单：查看菜品价格
  - 厨房：升级厨房设施
  - 升级：升级店铺设施
  - 员工：雇佣员工

### 升级系统
- **餐桌数量**：可接待更多顾客
- **翻台速度**：加快顾客用餐
- **厨房效率**：加快烹饪速度
- **等候区**：容纳更多排队顾客

### 员工系统
- **小王(服务员)**：自动上菜
- **钱女士(收银员)**：自动结账
- **老李(厨师)**：加快烹饪
- **罗阿姨(保洁)**：自动打扫
